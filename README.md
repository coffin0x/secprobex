# SecProbeX

SecProbeX is a comprehensive security testing tool designed to perform various vulnerability scans including subdomain enumeration, path discovery, and different types of injections and script injections.

## Features

- **Subdomain Finder**: Enumerates subdomains using a set of predefined payloads.
- **Path Finder**: Identifies valid paths on a given domain.
- **LFI (Local File Inclusion)**: Tests for Local File Inclusion vulnerabilities.
- **SQLi (SQL Injection)**: Tests for SQL Injection vulnerabilities.
- **XSS (Cross-Site Scripting)**: Tests for Cross-Site Scripting vulnerabilities.
- **SSTI (Server-Side Template Injection)**: Tests for Server-Side Template Injection vulnerabilities.
